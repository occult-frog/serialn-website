Install serialn

serialn is built for Macs with an Apple Silicon chip. You can use Method 1 to install it on Windows 11 but there is no guarantee it will work properly.

Method 1:

  1. Install python
  2. Go to github.com/occult-frog/serialn
  3. Click the green code button at the top and click "Download ZIP"
  4. Unzip the file you just downloaded
  5. Open terminal and navigate to the unzipped folder
  6. Run

    pip3 install -e .

  7. Now you can open a new terminal window and run commands. You don't need to add the "./serialn" prefix as in the example videos.

Method 2:

  1. Go to github.com/occult-frog/serialn
  2. Click on serialn-v1.0.0 under Releases which towards the right.
  3. In the page that opens download the serialn-v1.0.0.zip file.
  4. Unzip the downloaded file.
  5. In a terminal window navigate to the directory the serialn file is in.
  6. Now you can run commands. Just be sure to add the "./serialn" prefix before every command as in the example videos.
