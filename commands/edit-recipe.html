edit-recipe

Allows you to edit a recipe. Don't pass the recipe name as [RECIPENAME].csv. Pass it without the extension. If the recipe name argument is left blank then a list of recipes with a number next to them will be shown. Choose a number from the list to edit that recipe.

SYNTAX

edit-recipe [RECIPENAME] [OPTIONS]

OPTIONS

Flag Type Description Default
--ing, -i int Serial number of the ingredient to be edited -
--step, -s int Serial number of the step to be edited -
--note, -n int Serial number of the note to be edited -

EXAMPLES

Edit ingredient 2 and step 1 of recipe1

edit-recipe recipe1 -i 2 -s 1

Edit ingredient 4, step 2, and note 1 of recipe1

edit-recipe recipe1 -i 4 -s 2 -n 1

Edit note 5 of recipe1

edit-recipe recipe1 -n 5

Edit step 3 and note 2 of the recipe chosen from the list

edit-recipe -s 3 -n 2
